Mobile CMS

Start with Mobile CMS

消息佈告、門市搜尋、線上型錄、品牌導覽等即時的行動資訊，讓使用者隨查隨知，擺脫傳統紙本與網頁瀏覽的不便！

提升瀏覽體驗 20 - 60% 揮別傳統紙本與行動網頁的瀏覽不便，發揮 APP 原生操作的習慣優勢，提供使用者更佳的服務體驗。
促成來店率 3 - 11 % 在外也能透過 APP 掌握最新優惠、活動情報，還可透過 LBS 定位搜尋最近門市，連結 Google / Apple Map 導航路徑。
降低成本 10 - 35% 不再需要印製大量的紙本型錄，透過 APP 讓使用者線上查詢各式資訊，替企業省下龐大的印刷與寄送費用。
消息佈告傳達最新消息與情報
可應用在首頁輪播廣告、蓋版情報、最新消息等，並能設定不同類別。
門市搜尋迅速找到最近店點
透過地圖查找附近店點，並顯示門市營業時間、電話等相關資訊。
線上型錄便利又能降低成本
直接瀏覽型錄，減少印刷成本與郵寄費用，還可切換單品介紹模式、連結購物車。
專櫃導覽查找專櫃很容易
手機，就是逛街利器！根據樓層或分類，找到想逛的品牌，還能透過關鍵字查詢。
商品瀏覽隨時隨地逛商品
後台上稿或串接既有購物系統，顯示各類商品資訊，讓使用者隨時想逛就逛。

內容發佈管理 - 上下架發佈輕鬆設定

發佈內容時可設定上下架期間，自動發佈訊息，不用守候在電腦前手動作業。

內容編輯管理 - 支援 HTML 編輯模式

支援 HTML 編輯模式，讓內容修改更為彈性，且內容能即時反應於前端。

權限管控機制 - 角色與帳號輕鬆管理

可配置角色並管控帳號，讓不同單位或職務的人員各司其職，方便管理。

Know more about WishCMS

常見問題模組 透過後台編輯常見問答集，讓使用者可於 APP 得到常見問題的答案，減輕客服人員負擔。
菜單瀏覽模組 將餐廳的菜單資訊放到 APP 中，並可進行分類管理，讓使用者方便查找到想要的內容。
社群分享模組 將指定訊息傳至社群網路與好友分享，例如常用的 Facebook / LINE / WeChat ... 等。

行動時代的資訊傳達與版面呈現，不再侷限於紙本與傳統媒體您需要的是方便瀏覽且符合行動操作習慣的 CMS x APP協助您的企業、品牌提供顧客最即時的情報！
