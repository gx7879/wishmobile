Mobile POS

Start with Mobile POS

信用卡刷卡、第三方支付、發票列印、票券核銷等多種門市營銷服務，小小一台匯聚所有功能，讓門市營銷變得更輕巧、更智慧！

多元支付更便利 支援信用卡刷卡、各種第三方支付，相關認證齊全，只要一台機器就能完成多種金流支付，再也不需要滿桌的刷卡機、支付機。
發票列印一機搞定 手持式智慧裝置，整合多樣化硬體設備，包含刷卡機與熱感列印機，支援電子發票列印、商品明細列印、信用卡簽單列印、優惠條碼列印。
核銷票券最即時 前後鏡頭搭配軟體辨識條碼，無論是一維條碼或是二維條碼，都能快速解讀；並可即時連線至票券系統進行查詢、核銷作業。
多元支付不再堆滿各種支付機
只要一個手持智慧裝置，就能解決門市端每種支付都需要一種硬體設備的問題。
收銀服務行動收銀又節省空間
支援簡易手輸金額結帳、商品掃碼結帳、專櫃類別結帳，整合收銀與支付作業。
票券核銷掃碼核銷票券很便利
透過鏡頭掃描 QR Code / BarCode 並連線至票券系統完成核銷作業。
發票列印熱感列印很迅速
結合熱感列印設備，可列印電子發票 / 商品明細 / 信用卡簽單 / 優惠券條碼 ... 等。

行動時代的收銀與支付作業，不再需要佔空間的固定式電腦設備您需要的是一機成型、整合刷卡與熱感列印的行動裝置協助您的企業、品牌提供更靈活的門市營銷服務！

威許移動 ( WishMobile, Inc. ) 與合作夥伴發展一機多用的 Mobile POS 服務，透過行動化特性提供靈活且智慧的門市營銷應用 ( 信用卡刷卡、第三方支付、票券核銷、發票列印、智慧收銀 ... 等 )，無論您的企業或品牌是否已經具備收銀 POS 系統，我們都可以透過 Mobile POS 擴充相關服務，不僅不佔用您寶貴的門市空間，更能跟上時代潮流提供新穎體驗！
