Mobile Wallet

Start with Mobile Wallet

行動儲值卡、行動會員錢包、行動票券匣等多種支付與核銷方式，讓行動支付變得更容易，提供更友善的購物體驗！

提升購買意願 支付時不需要輸入信用卡號，或進行簡訊 OTP 驗證，透過自有的電子錢包支付，讓支付流程變得更簡單，降低使用者操作難度，提升購買意願。
自有錢包忠誠經營 綁定儲值卡、信用卡至會員帳戶，未來消費時，只要掃瞄虛擬會員卡，累點、折抵、支付一次到位，再也不需要攜帶實體卡片。
不佔用門市空間 透過既有 POS 機串接整合支付 API 接口，就能實現信用卡刷卡、第三方支付，無須放置多台刷卡機、支付機，節省寶貴的門市空間。
行動會員錢包折抵支付一次到位
綁定會員與信用卡，未來消費只要出示 APP 交易條碼，就能連結信用卡扣款。
行動儲值卡儲值與扣款很便捷
不用隨身攜帶儲值卡，透過行動條碼就能完成儲值與扣款，還能透過線上刷卡儲值。
行動票券匣票卡通通虛擬化
將飲料寄杯券、現金抵用券、優惠折扣券通通放入行動票券匣，開啟條碼即可核銷。

行動時代的支付與核銷方式，不再需要實體票卡塞滿錢包
您需要的是結合支付作業與會員忠誠的行動錢包
協助您的企業、品牌提供更友善的支付體驗！
